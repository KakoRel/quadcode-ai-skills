# Quadcode.ai Skills & Projects

Welcome to the Quadcode.ai Skills repository. This project demonstrates how specialized AI agents can automate complex, multi-step tasks through structured workflows and generative capabilities.

---

## 📂 Repository Structure

### 🔧 [skills/](./skills/) — The "Brains"
This folder contains the core logic and definitions for our AI agents. Each skill is a production-ready methodology that an agent follows to achieve a specific goal.
- **[Flow Builder](./skills/skill_1_flow_builder/)**: Designs multi-agent AI workflows and outputs technical YAML specifications.
- **[Campaign Kit](./skills/skill_2_campaign_kit/)**: Generates full marketing campaigns, including copy, images, video, and audio.

### 🚀 [projects/](./projects/) — The "Results"
This folder stores the artifacts and specifications generated by the skills. It serves as a gallery of real-world outputs.
- **[Summit Roast Campaign](./projects/campaign_kit_test/)**: A complete marketing package (images, video, email, manifest).
- **[Socratic Tutor Spec](./projects/algebra_tutor/)**: A technical blueprint for an AI math tutoring system.

---

## 🛠 How to Use

### 1. Explore Skills
Navigate to the `skills/` directory to understand the rules, workflows, and architecture behind each capability. You can load these into the Quadcode.ai platform using their aliases:
- `flow_builder`
- `campaign_kit`

### 2. Review Generated Projects
Check the `projects/` directory to see the quality and structure of the outputs. These files are ready for implementation or deployment.

### 3. Learn the Workflow
Read the [screencast_script.md](./screencast_script.md) to see a step-by-step demonstration of how these skills transform a simple brief into a professional project.
