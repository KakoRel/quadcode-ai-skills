# Flow Builder — Thumbnail Concept

## Generation Prompt

```
A futuristic network diagram on a deep dark navy (#0B0E1A) background. 
Six luminous circular nodes arranged in a horizontal flow — left to right — connected by glowing neon blue (#00D4FF) and purple (#A855F7) bidirectional data lines. 
Each node is a minimalist tech-circle with a subtle translucent glass effect, containing a simple abstract icon: 
a search/magnifier, a gear, a branching fork, a diamond/gem, a checkmark shield, and a document. 
Faint particle-like dots drift along the connection lines. 
Soft volumetric glow behind each node. 
No text. 
Cinematic lighting, octane render quality, isometric slight angle, 
professional clean sci-fi aesthetic, high contrast, sharp focus.
```

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#0B0E1A` | Deep dark navy — space/network feel |
| Primary node accent | `#00D4FF` | Cyan electric blue — active connections |
| Secondary accent | `#A855F7` | Vivid purple — alternate nodes, depth |
| Node fill | `#1A1F35` | Translucent glass dark fill |
| Glow/highlight | `#60EFFF` | Soft glow on active elements |
| Data line | `#2DD4BF` | Teal — secondary connection paths |

## Style Notes

- **Mood**: Professional, technical, futuristic — not gaudy or game-like
- **Composition**: 6 nodes in a horizontal S-curve or slight arc, creating visual flow
- **Depth**: Subtle isometric tilt (15°) to give the 2D diagram a 3D feel
- **Lighting**: Soft volumetric glow behind nodes, simulated rim light on circles
- **Texture**: Clean, no noise/grain — this represents an architecture tool
- **Typography**: No text — let the visuals communicate "network/system/flow"
- **Detail level**: Medium — readable at 1024x1024 and at catalog card size (~200px)

## Composition Guidance

- Canvas: 1024x1024 square
- Nodes: ~80px diameter each
- Spacing: Even horizontal distribution with slight Z-depth variation
- Node 1 (left) slightly smaller (input), Node 6 (right) slightly larger (output)
- Connection lines: 2px width with animated data-dot look (dots mid-traversal)
- Overall shape: Gentle upward arc from left to right, then a downward settle
- Negative space: ~15% margin around the network on all sides
- Focus zone: Center 70% of canvas — nodes 2-5 in the sharpest focus

## Output Spec

- **Format**: WebP
- **Dimensions**: 1024x1024
- **Intent**: Catalog avatar / thumbnail grid item
- **Style taxonomy**: Sci-fi / tech / diagram / network / minimal
